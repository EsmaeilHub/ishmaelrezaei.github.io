HYBRID: HYper‑reduced Basis Reduction via Interactive Decomposition

Real-world datasets often consist of a large number of features or variables, with billions of samples, making it computationally expensive to manipulate such high-dimensional data algebraically. Dimension reduction techniques are used to transform high-dimensional data into a lower-dimensional approximation, improving the accuracy of analysis. These methods can be linear, such as Principal Component Analysis (PCA) and Singular Value Decomposition (SVD), or nonlinear, focusing on capturing local relationships between data points. This paper introduces HYBRID, a linear method inspired by the reduced basis method (RBM), designed to accelerate numerical simulations of Partial Differential Equations. HYBRID's innovative approach and offline-online decomposition lead to faster computations and low-dimensional vectors accompanied by error estimators. It provides an efficient solution for high-dimensional data reduction, and you can find further details and code on the provided GitHub repository.

In this project, we developed a high-speed dimensional reduction algorithm designed for efficiently handling massive matrices. Our work involved enhancing the Reduced Basis Dimension (RBD) algorithm by incorporating numerical solutions for partial differential equations. We are currently preparing to release the code for the HYBRID algorithm following its paper publication.

Figure 1. These figures show the reconstructed images achieved by reducing the number of columns to 25 (left), 51 (middle), and 170 (right) from an original 512.

In the meantime, we offer access to the RBD algorithm, complete with two practical examples: image compression and video compression. It's important to note that the primary goal isn't limited to the application of HYBRID or RBD in video/image compression; instead, it focuses on using images and videos to effectively illustrate the efficiency of these algorithms.
